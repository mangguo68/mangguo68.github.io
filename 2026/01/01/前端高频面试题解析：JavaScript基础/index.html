前端高频面试题解析：JavaScript基础

一、闭包

题目：什么是闭包？闭包有什么优缺点？

答案：
闭包是指有权访问另一个函数作用域中变量的函数。

解题思路：

  1. 闭包的形成条件：函数嵌套 + 内部函数引用外部函数的变量 + 内部函数在外部可访问
  2. 闭包的特性：延长变量的生命周期
  3. 闭包的应用场景：模块化、防抖节流、函数柯里化

延伸知识点：

  • 闭包可能导致内存泄漏，因为它会持有对外部变量的引用
  • 解决方法：在不需要使用闭包时，手动解除引用

二、原型链

题目：什么是原型链？如何理解 JavaScript 的原型继承？

答案：
原型链是 JavaScript 实现继承的机制，每个对象都有一个原型对象，原型对象又有自己的原型，形成一个链式结构。

解题思路：

  1. 每个函数都有一个 prototype 属性，指向它的原型对象
  2. 每个对象都有一个 __proto__ 属性，指向它的构造函数的原型对象
  3. 当访问一个对象的属性时，如果对象本身没有该属性，会沿着原型链向上查找
  4. 原型链的终点是 null

延伸知识点：

  • ES6 的 class 语法是原型继承的语法糖
  • Object.create() 可以直接指定对象的原型
  • instanceof 运算符通过原型链判断对象类型

三、异步编程

题目：请解释 JavaScript 的异步编程模型，以及 Promise、async/await 的工作原理。

答案：
JavaScript 是单线程语言，通过事件循环实现异步编程。Promise 是用于处理异步操作的对象，async/await 是基于 Promise 的语法糖。

解题思路：

  1. 事件循环：调用栈、宏任务队列、微任务队列
  2. Promise 的三种状态：pending、fulfilled、rejected
  3. async 函数返回一个 Promise 对象
  4. await 关键字会暂停 async 函数的执行，等待 Promise 解决

延伸知识点：

  • 宏任务：setTimeout、setInterval、I/O 操作
  • 微任务：Promise.then、MutationObserver、process.nextTick
  • Promise.all、Promise.race、Promise.allSettled 的使用场景

四、防抖和节流

题目：什么是防抖和节流？它们的应用场景是什么？

答案：
防抖和节流都是用于优化频繁触发的事件处理函数的技术。

解题思路：

  1. 防抖：在事件触发后等待一段时间再执行，如果在等待期间事件再次触发，则重新计时
  2. 节流：限制事件处理函数的执行频率，在一定时间内只执行一次

延伸知识点：

  • 防抖的应用场景：搜索框输入、表单验证、窗口 resize
  • 节流的应用场景：滚动事件、鼠标移动、游戏中的动画

五、深拷贝和浅拷贝

题目：什么是深拷贝和浅拷贝？如何实现深拷贝？

答案：
浅拷贝只复制对象的引用，深拷贝复制对象的所有层级。

解题思路：

  1. 浅拷贝方法：Object.assign()、展开运算符 {...obj}、Array.prototype.slice()
  2. 深拷贝方法：JSON.parse(JSON.stringify())、递归拷贝、第三方库（如 lodash.cloneDeep）

延伸知识点：

  • JSON.parse(JSON.stringify()) 不能处理循环引用、函数、Symbol 等
  • 递归拷贝需要处理各种数据类型和循环引用

六、总结

JavaScript 基础是前端面试的重点，掌握闭包、原型链、异步编程、防抖节流、深拷贝和浅拷贝等核心概念，对于通过前端面试至关重要。

在准备面试时，不仅要记住这些概念的定义，还要理解它们的工作原理和应用场景，这样才能在面试中从容应对各种问题。
